## Project Setup

### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# Vite build
$ npm run build

# Then
$ npm run make

# or
$ npm run publish
$ npm run publish:all # Github Publisher
```
